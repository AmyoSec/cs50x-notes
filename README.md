# cs50x-notes
Weekly theoretical notes in Markdown format covering Harvard's CS50x lectures. Strictly adhering to the academic honesty policy.

---

## Overview

This repository contains my personal study notes, conceptual breakdowns, and syntax guides for **Harvard University's CS50x: Introduction to Computer Science**. 

The main objective is to document my learning journey from the ground up, consolidate core computer science principles, and maintain a clean, readable reference for the community.

---

## Repository Structure

Notes are organized systematically by week, following the official CS50x curriculum:

*   `week00-scratch.md` - Computational Thinking, Algorithms, and Pseudocode.
*   `week01-c.md` - Source Code, Compilers, Data Types, and Logic in C.
*   `week02-arrays.md` - Memory Layout, Compiling, Debugging, and Arrays.
*   `week03-algorithms.md` - Linear/Binary Search, Sorting Algorithms, and Big O Notation.
*   `week04-memory.md` - Hexadecimal, Pointers, Dynamic Memory Allocation, and File I/O.
*   `week05-data-structures.md` - Stacks, Queues, Linked Lists, Trees, and Hash Tables.
*   `week06-python.md` - Syntax Transition, High-level Abstractions, and Scripting.
*   `week07-sql.md` - Relational Databases, Queries, and Data Management.
*   `week08-html-css-javascript.md` - Web Fundamentals, DOM Manipulation, and Frontend.
*   `week09-flask.md` - Web Frameworks, MVC Architecture, and Server-side Logic.
*   `week10-final-project.md` - Capstone Project, Software Engineering, and The End.
